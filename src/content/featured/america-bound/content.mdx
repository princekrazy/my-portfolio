# AmericaBound

A lightweight, filterable university discovery platform designed to help international students find affordable and accessible U.S. universities.

---

# Overview

AmericaBound was created from a personal challenge:

Finding affordable universities in the U.S. as an international student is difficult due to:

- Fragmented tuition data
- Inconsistent admission systems
- Lack of proper filtering tools
- Slow, bloated existing platforms

This project centralizes key decision-making data into a **fast, simple, and mobile-friendly interface**.

---

# The Problem

International students face several challenges:

- Tuition information is scattered or unclear
- Admission cycles vary widely across universities
- No easy way to filter by affordability or acceptance rate
- Existing platforms are heavy and slow
- Poor optimization for low-end devices

This makes university research overwhelming and inefficient.

---

# The Solution

AmericaBound focuses only on essential decision-making data:

- Tuition (international pricing)
- Acceptance rate
- State location
- Rolling admissions
- CSS financial aid indicator

### Example Data Structure

```json
{
  "id": 1047,
  "university_name": "SUNY Fredonia",
  "logo": "fredonia",
  "state": "New York",
  "international_price": 21000,
  "CSS": "no",
  "acceptance_rate": 84,
  "rolling_admission": "yes"
}
```

# Key Features

## University Discovery

Browse a curated list of U.S. universities with simplified decision data.

---

## Smart Filtering

Users can filter by:

- Budget (international tuition)
- Acceptance rate
- State
- Admission type (rolling vs fixed)

---

## Shortlisting System

Users can save universities they are interested in.

- Stored in Local Storage
- Persistent across sessions
- No login required

---

## Lightweight Design Philosophy

The application was built with accessibility in mind:

- Optimized for low-end devices
- Fast load times on slow networks
- No backend dependency
- Works without authentication

# Design Philosophy

This project prioritizes **accessibility over complexity**.

Many international students face:

- Limited device performance
- Expensive or unstable internet
- High data costs

---

## Evolution of the Project

Originally built as a **React Native mobile app** and published on the Google Play Store.

However:

- Platform restrictions limited reach
- Distribution barriers reduced accessibility

So the project was rebuilt as a:

> Web-first application

Now it is accessible on any device without installation.
