# Uber Clone Frontend

A modern ride-hailing frontend built with **React** and **MapLibre GL**, designed to simulate a production-grade Uber-like system with real-time driver-rider interaction, persistent ride state, and live geolocation tracking.

---

## Overview

This project demonstrates a full ride-hailing experience including:

- Real-time map rendering
- Live driver tracking
- Ride lifecycle management
- Persistent session recovery
- Role-based flows (Rider & Driver)
- Payment integration

It focuses heavily on **real-time frontend architecture** and **state synchronization challenges**.

---

## Why MapLibre?

Instead of proprietary mapping solutions, this project uses **MapLibre GL**.

### Benefits

- Fully open source
- No vendor lock-in
- High-performance vector rendering
- Production-ready mapping engine
- Strong community support

```js
import maplibregl from "maplibre-gl";
```
