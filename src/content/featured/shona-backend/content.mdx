# Shona AI Translator (Backend)

A FastAPI backend that powers an AI-based translation system converting English to Shona.

It handles translation requests, stores user history, and provides a clean REST API for frontend integration.

---

# 🌍 Live API

https://shona-translation-ai.onrender.com

---

# ⚡ Features

## 🔁 REST API for Translation

Provides a simple endpoint for translating text from English to Shona.

---

## 🗄 Translation History Storage

- Stores translations in **SQLite**
- Keeps record of all user requests
- Enables history retrieval for frontend UI

---

## 🧠 Clean Modular Architecture

The backend is structured for scalability and maintainability.

---

## 🔌 CORS Enabled

Allows secure communication with frontend applications.

---

## 📦 Lightweight & Fast

Built to be minimal, fast, and easy to deploy.

# 🧱 Tech Stack

- FastAPI

# 🔌 API Endpoints

## 📝 Translate Text

```
POST /translate
```

### Request

```json
{
  "text": "Hello, how are you?"
}
```

### Response

```json
{
  "id": 1,
  "original": "Hello, how are you?",
  "translation": "Mhoroi, makadini"
}
```

---

## 📜 Get History

```
GET /history
```

Returns a list of previously translated texts.

# 🚀 Run Locally

## 1. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

---

## 2. Install dependencies

```bash
pip install -r requirements.txt
```

---

## 3. Start server

```bash
uvicorn main:app --reload
```

Server runs at:

http://127.0.0.1:8000

5. Response is returned to frontend
