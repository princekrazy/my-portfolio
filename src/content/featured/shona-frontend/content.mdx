# Shona AI Translator (Frontend)

A modern AI-powered language translation frontend built with **Next.js**, designed to translate English text into **Shona** using a FastAPI backend.

---

# Live Demo

https://shona-frontend.vercel.app/

---

# Overview

This application provides a clean, chat-style interface for interacting with an AI translation system.

It focuses on:

- Simple user experience
- Fast real-time translation
- Clean conversational UI
- Seamless backend communication

---

# Features

## Chat-Style Translation UI

Users interact with the translator like a chat system:

- Input English text
- Receive Shona translation instantly
- Clean conversational layout

---

## Real-Time API Communication

The frontend communicates directly with a FastAPI backend to generate translations.

- Low-latency requests
- Instant response rendering
- Efficient fetch handling

---

## Translation History Sidebar

- Stores previous translations
- Allows users to revisit past queries
- Improves usability for repeated translations

---

## Copy to Clipboard

Users can easily copy translated text with one click.

---

## Responsive Design

Optimized for:

- Mobile devices
- Tablets
- Desktop screens

Ensures smooth experience across all platforms.

---

# Tech Stack

- Next.js (React framework)
- JavaScript (ES6+)
- Tailwind CSS
- Fetch API
- REST API integration

# Backend Integration

This frontend connects to a **FastAPI backend**:

https://shona-translation-ai.onrender.com

---

## API Endpoints

### Translate Text

```
POST /translate
```

### Fetch History

```
GET /history
```
