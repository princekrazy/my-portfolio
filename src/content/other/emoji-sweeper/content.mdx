# Emoji Sweeper

A Visual Studio Code extension that removes emoji characters from source code quickly and efficiently.

It helps clean copied code, enforce formatting standards, and prepare files for production environments.

---

# Overview

Emoji Sweeper provides a simple set of commands to remove emojis from:

- Entire files
- Selected text
- Specific line ranges

It is designed to be fast, lightweight, and developer-friendly.

---

# Features

## Remove Emojis from Whole File

Cleans the entire active file in one command:

```
Remove Emojis: Whole File
```

---

## ️ Remove Emojis from Selection

Only cleans highlighted code:

```
Remove Emojis: Selection
```

---

## Remove Emojis from Line Range

Target specific sections of a file:

```
Remove Emojis: Line Range
```

Users specify start and end line numbers.

---

## Lightweight & Fast

- Minimal performance overhead
- Instant text transformation
- No external dependencies

# Installation

## Marketplace Install

Install directly from the Visual Studio Code Marketplace.

---

## Manual Install (.vsix)

1. Download the latest `.vsix` file
2. Open Visual Studio Code
3. Open Command Palette:

```
Ctrl + Shift + P
```

4. Run:

```
Extensions: Install from VSIX...
```

5. Select the downloaded file
