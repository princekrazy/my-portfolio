# The Empty Library

An interactive React-based simulation game that explores the real-world consequences of book banning in schools.

---

# Overview

The Empty Library is an interactive decision-based simulation where the player acts as a school administrator making choices about whether to ban or keep challenged books.

Each decision influences:

- School environment
- Community reaction
- Narrative outcome

The goal is to simulate how policy decisions can have unintended consequences.

---

# Motivation

This project was originally created as a classroom assignment focused on raising awareness about book banning in schools.

Instead of writing a traditional essay, I chose to:

> Build an interactive simulation using React.

The goal was to communicate the topic in a more engaging and experiential way.

---

# Core Idea

Rather than explaining book banning through text, the project allows users to:

- Make decisions
- Experience consequences
- Observe system-wide effects

This transforms an abstract topic into an interactive system.

# Features

## Interactive Decision System

Players choose whether to:

- Ban books
- Keep books

Each decision affects the simulation outcome.

---

## Book System

Each book includes:

- Themes / tags
- Challenge reasons
- Narrative consequences

---

## ️ Dynamic Rule System

Game logic evolves based on decisions:

- Banning certain categories affects others
- System responses adapt over time
- Decisions have cumulative impact

---

## Escalating Consequences

Player actions can trigger:

- Parent complaints
- Community backlash
- School instability

---

## Multiple Endings

Different outcomes based on decisions:

- **“Empty Library”** → over-banning outcome
- **“Empty School”** → community collapse
- Balanced or neutral outcomes depending on choices

---

## Difficulty Modes

- Short simulation mode
- Full extended simulation mode
