# Vault Manager Desktop

A desktop cash vault management application built with **React, Tauri, Rust, and SQLite**.

Vault Manager is designed for secure offline financial tracking, including deposits, withdrawals, ledger management, and automated reporting.

---

# Overview

This application runs entirely on the user's machine with no internet dependency.

It provides a complete system for:

- Cash tracking
- Transaction logging
- Balance calculation
- Report generation

---

# Features

## Deposits

Users can record incoming cash:

- Depositor name
- Amount
- Notes
- Manual date selection
- Auto-suggestions from previous entries

---

## Withdrawals

Users can record outgoing cash:

- Recipient name
- Amount
- Notes
- Manual date selection
- Auto-suggestions from previous entries

---

## Ledger System

A full transaction history view:

- View all transactions
- Edit entries
- Delete entries
- Sort by newest first

---

## Filtering System

Advanced filtering options:

- Filter by name
- Minimum amount
- Maximum amount
- Start date
- End date

---

## Balance Dashboard

Automatically calculates:

- Total Deposits
- Total Withdrawals
- Net Balance

Formula:

```
Net Balance = Total Deposits - Total Withdrawals
```

---

## Reporting

Export financial data:

- Excel export (SheetJS)
- PDF export (jsPDF)

---

## Offline-First Design

Vault Manager operates completely offline:

- No internet required
- No cloud storage
- No external servers
- Fully local SQLite database

# Database Schema

## Transactions Table

```sql
CREATE TABLE transactions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    type TEXT NOT NULL,
    name TEXT NOT NULL,
    amount REAL NOT NULL,
    note TEXT,
    transaction_date TEXT NOT NULL
);
```
